# paper ディレクトリ

このディレクトリは、論文化用の原稿下書きを置くための作業領域です。

## ファイル
1. `JSET_実践報告_下書き.md`  
JSET投稿を想定した実践報告の原稿下書き（日本語）。
2. `JSET_投稿言語確認_2026-02-27.md`  
JSET公式規程に基づく、英語投稿可否の確認メモ。
3. `JSET_ショートレター_即投稿テンプレ_結果差し替え版.md`  
結果値を差し替えるだけで使える、ショートレター向け原稿テンプレート。
4. `ITEL_Short_Paper_Ready_Template.md`  
ITEL Short Paper向けの英語テンプレート（結果差し替え式）。
5. `ITEL_Short_Submission_Checklist_2026-02-27.md`  
ITEL提出時の実務チェックリスト（規程対応）。
6. `ITEL_Short_4Page_Draft_v1.md`  
タイトル・著者・対象Nを先に埋めた、4ページ想定の圧縮ドラフト。

## 使い方
1. 下書き内の `__` を実データで置換する。  
2. JSETの最新テンプレート（Word/B5）へ転記する。  
3. 匿名査読版では、学校名・個人名・プロジェクト識別情報を削除する。
